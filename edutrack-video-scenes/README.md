# EduTrack Brand Animation — Scenes

This folder contains the individual scene components for the **EduTrack** brand animation video.

Each scene is a self-contained React + Framer Motion component built with:
- Dark cinematic background
- Gold & blue glow / glassmorphism aesthetic
- Hind Siliguri font for Bangla typography

## Scenes

| File | Duration | Description |
|------|----------|-------------|
| `Scene1.tsx` | 5 seconds | Logo reveal — dark screen → golden light → SVG logo draw-in → Bangla tagline fade |

## Adding a New Scene

1. Create `Scene2.tsx` (or the next number) in this folder.
2. Copy it into `artifacts/edutrack-video/src/components/video/video_scenes/` in the Replit project.
3. Register it in `VideoTemplate.tsx` under `SCENE_DURATIONS` and `SCENE_COMPONENTS`.
